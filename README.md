# Five Peaks Coffee | Premium Barista Catering

Professional barista coffee catering for corporate events, conferences, and festivals across the UK. Precision in every pour.

## Features

- **Specialty Coffee**: Ethically sourced beans and expert baristas.
- **Bespoke Service**: Tailored menus and branding for corporate events and weddings.
- **High Capacity**: Built for high-volume festivals and large-scale conferences.
- **Real-time Booking**: Integrated booking wizard for event inquiries.
- **Case Studies**: Showcase of recent high-profile projects (Recent Peaks).

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4
- **Animations**: Motion (Framer Motion)
- **Icons**: Lucide React
- **Backend/Database**: Firebase (Firestore & Authentication)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd five-peaks-coffee
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file based on `.env.example` and add your Gemini API key if using AI features.

4. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create a production build:
```bash
npm run build
```
The output will be in the `dist/` directory.

## Firebase Configuration

This project uses Firebase. The configuration is stored in `firebase-applet-config.json`. For a new deployment, you will need to replace the values in this file with your own Firebase project credentials.
